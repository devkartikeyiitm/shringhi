# Shringhi - Construction Website

A modern, professional construction website built with Next.js and React. Shringhi showcases a professional construction company's services with a clean, responsive design.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern Stack**: Built with Next.js for optimal performance
- **TypeScript Support**: Full TypeScript support for type safety
- **CSS Styling**: Clean and maintainable CSS styling
- **Professional Layout**: Designed to showcase construction services effectively

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: JavaScript & TypeScript (76.7% JS, 17.5% TS)
- **Styling**: CSS (5.8%)
- **Package Manager**: npm

## Getting Started

### Prerequisites

- Node.js (v14 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone https://github.com/devkartikeyiitm/shringhi.git
cd shringhi
```

2. Install dependencies:

```bash
npm install
```

### Running the Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
